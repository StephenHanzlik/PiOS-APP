PiOS Media

PiOS is an Ionic iOS app that gives you the ability to stream HD video from a secure PiOS Media server directly to your iOS devices. The PiOS Media Server is built using Raspberry Pi 3, NodeJS, Express, Knex and PostgreSQL. The PiOS Media Server never connects to the internet and is secured from web based attacks. The PiOS Media Server creates a local area network which the PiOS app connects to and uses this connection to stream videos in HD quality.
